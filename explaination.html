Renderer: WEBGL2

Description:
The renderer is of an MousePad with an RGB strip around it.

How the MousePad is rendered:
For rendering the texture on the mousepad itself I have created 8 polygons.
1 for the lower left texture. 1 for the upper right texutre. 3 polygons each forming the two textures in the middle.
I have assigned the respective color for each polygon.

How the strip is created:
I have created an algorithm that starts from the upper left vertex of the mousepad moving right then down then left then up and during this movement it is essentially creating a small set of triangles around the mousepad which have colors assigned to them.

How the strip is animated:
For each of the modes I have simply created different functions that assign colors to the triangles around the mousepad. For each animation I have added a sleep command in the render function.

Breathe:
Changes the color of all strips around the mousepad from black to color in 4 stages.

Rainbow:
Changes the left strip, then assings the same color to botom strip, then right strip then top strip while also changing colors of previous strips.

Cycle:
Changes color around mousepad in cycle fashion.

Disco:
Changes color around mousepad in fast motion.

Speed Bar:
Changes the value passed to sleep function.
